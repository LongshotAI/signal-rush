03 — Architecture

Engine-first.
Deterministic. Testable.

The engine has zero I/O dependencies. It steps pure state. One core, read by many surfaces. That's how you ship games that work everywhere without rewriting everything.

⚙️

Pure Core Engine

940 lines of deterministic game logic. No Math.random() in engine code — seeded RNG only. Reusable by browser, CLI, and future mobile renderers.

🧪

229+ Tests

Mechanics, smoke, persistence, compact renderer, embedded widget, economy integration, replay verification. Fresh-clone proofed.

🔌

Embed-Ready

A 6–10 row widget that lives in the bottom band of any agent CLI. Never steals keystrokes. ANSI scroll-region. Idempotent lifecycle.

💰

Economy Layer

SQLite ledger, ad-funded rewards pool, ppq.ai redemption. Credit diffing in the event bridge — no double-counting, never blocks gameplay.
