# Signal Rush MVP

Small web-first prototype for the `Signal Rush` concept: a reflex survival game with placeholder monetization surfaces and platform-credit rewards.

## Run

Open [index.html](./index.html) directly in a browser, or serve the folder with any static file server.

## MVP Scope

- One playable survival arena
- Keyboard movement and dash
- Score, combo, integrity, and credit reward loop
- Passive ad placeholder in HUD
- Interstitial placeholder on run end
- Placeholder art rendered in canvas/CSS instead of production assets

## Next Build Step

Port the loop into React + Phaser once the team wants componentized UI, scene management, and mobile adaptation.
